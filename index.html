    Hallo, mijn naam is Mathijs Jonker en ik ben 18 jaar oud.

    Ik wil deze opleiding volgen omdat ik meer wil leren over programmeren. Mijn interesse in software is begonnen toen ik 4 jaar geleden Minecraft ben gaan spelen. Doordat de game makkelijk te modificeren is kreeg ik de kans om een beetje Java te oefenen. Dat was voor mij genoeg om te beseffen dat ik coderen ook leuk vind. Nu volg ik al een paar maanden cursussen voor HTML CSS en sinds kort ook voor Python bij Hacklab in Leeuwarden. Daarbij maak ik gebruik van Visual studio en IDLE om te programmeren.

    Mijn kwaliteiten zijn dat ik sociaal ben, daardoor kan ik makkelijker samenwerken met anderen. Er is namelijk veel teamwork nodig om goede programma’s te schrijven. Ik kan ook goed naar anderen luisteren. Dat is erg belangrijk als je een programma voor iemand maakt. Anders zou je niet weten wat diegene wil. Als iets niet goed lukt of moeilijk is geef ik ook niet snel op. Een goed doorzettingsvermogen is essentieel voor een programmeur.

    Voor het maken van websites maak ik gebruik van HTML in combinatie met CSS. Maar met HTML + CSS kun je alleen statische websites maken, dat houdt in dat een gebruiker de pagina inhoud alleen maar kan bekijken.

    Je zou er bijvoorbeeld ook Javascript bij kunnen gebruiken om een dynamische website te maken, bijvoorbeeld een blog waarbij de gebruikers zelf een reactie kunnen nalaten.

    Javascript is niet de enige programmeertaal waarmee je dat kan doen. Python, Ruby, PHP en Perl zijn allemaal talen waarmee je een dynamische website kan maken. Iedere taal is weer geschreven met een ander doeleinde en daardoor anders van elkaar. Er zijn dus genoeg verschillende talen om te beslissen welke het beste bij je project past.

    Als ik een mobiele app moest maken zou ik voor Java kiezen om mee te programmeren, omdat het de officiële taal is voor Android-systemen. Naast Java wordt C++ ook veel gebruikt voor Android. Voor IOS-systemen wordt er vooral de programmeertaal Object-C gebruikt.

    Voor Windows apps zou ik Python gebruiken omdat die het makkelijkste is om mee te beginnen. Verder zou je ook Java, C#, en C++ kunnen gebruiken om applicaties te maken.

    Door dit verslag te maken heb ik veel geleerd over allerlei verschillende programmeertalen en ik hoop met deze opleiding nog veel meer te leren en mijn vaardigheden te ontwikkelen.
